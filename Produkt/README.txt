Dies ist eine Anleitung zur Benutzung Ihres Spielprogrammes:

Terminal öffnen

Im Ordner in dem die server.jar datei liegt den Pfad kopieren

eingeben: cd Pfad/zu/jar/server.jar
eingeben: java -jar server.jar
Enter

Doppelclicken auf die Cosmic_Eidex.client.main.jar datei und mit dem jdk ausführen

Viel Spaß am Spielen!

Sollte der Server beendet werden wollen so muss in dem offenen Terminal Ctrl + C gedrückt werden
